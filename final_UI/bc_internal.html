Release Gate 예시

            배포 차단 기준
          

          

            P0 안전성 실패 수 = 0
            JSON schema pass rate >= 99.5%
            핵심 질문 pass rate 하락 <= 1.0%p
            candidate pairwise win rate >= 52%
            RAG faithfulness 하락 <= 2.0%p
            결제/환불/개인정보 slice regression 허용치 이하
            p95 latency와 평균 비용 목표치 이하
          

        

      

      

        

          
테스트 케이스 유형

          LLM 회귀 테스트 6종 세트
        

        

          
Golden-set
운영 로그, 도메인 전문가 질문, 과거 장애 케이스 기반의 필수 통과 테스트입니다.

          
Behavioral
환불, 개인정보, 표현 변형 등 서비스가 지켜야 하는 행동을 capability x test type으로 검증합니다.

          
Pairwise
기준 모델과 후보 모델 답변을 가리고 비교해 어떤 케이스가 좋아졌고 나빠졌는지 봅니다.

          
Metamorphic
paraphrase, 입력 순서 변경 등 라벨이 어려운 상황에서 결과 불변성을 확인합니다.

          
RAG
retrieval quality, context faithfulness, answer quality를 분리해 원인을 추적합니다.

          
Safety
jailbreak, prompt injection, 개인정보 유출, 정책 우회를 별도 gate로 차단합니다.
