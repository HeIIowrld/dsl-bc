총점
RAG 대상은 ACC, COM, UTL, NAC, HAL 5개 지표를 사용하고, 비-RAG 대상은 UTL을 제외한 4개 지표를 100점으로 정규화합니다.

            
규칙 기반
토큰 overlap, 필수/금지 조건, 숫자/날짜/JSON 형식, 안전 패턴을 규칙 기반으로 점검합니다.

            
LLM 평가
선택한 채점 모델이 독립적으로 정확성, 완결성, 수치 정확성, 환각 억제를 평가하고 RAG 대상에서만 근거 활용을 반영합니다.

            
비교
동일 테스트셋과 동일 채점 방식 기준으로 모델, 프롬프트 변형, 채점 설정별 성능을 비교합니다.
